CPUе’ҢRAMеҚ•дҪҚ
CPUиө„жәҗд»ҘcpusдёәеҚ•дҪҚгҖӮе…Ғи®ёе°Ҹж•°еҖјгҖӮеҚ•дҪҚдёәmillicores пјҢдҪ еҸҜд»Ҙз”ЁеҗҺзјҖmжқҘиЎЁзӨәmiliгҖӮдҫӢеҰӮ100mпјҢ ж„ҸжҖқжҳҜдҪҝз”ЁеҚ•дёӘж ёеҝғзҡ„1/100гҖӮ1core=1000mпјҢиӢҘиҠӮзӮ№жҳҜ4cpuеҜ№еә”зҡ„дёә4*1000=4000mгҖӮ

1000m (milicores) = 1 core
100m (milicores) = 0.1 core

RAMиө„жәҗд»ҘbytesдёәеҚ•дҪҚгҖӮдҪ еҸҜд»Ҙе°ҶRAMиЎЁзӨәдёәзәҜж•ҙж•°жҲ–е…·жңүиҝҷдәӣеҗҺзјҖд№ӢдёҖзҡ„е®ҡзӮ№ж•ҙж•°пјҡ

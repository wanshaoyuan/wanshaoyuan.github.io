ironic-з®Җд»Ӣ

2017-05-08 openstack openstack иҜ„и®ә еӯ—ж•°з»ҹи®Ў: 1,059(еӯ—) йҳ…иҜ»ж—¶й•ҝ: 4(еҲҶ)

ironicз®Җд»Ӣ

д»Ӣз»Қ

openstackдёӯеҜ№зү©зҗҶжңәз®ЎзҗҶзҡ„з»„д»¶пјҢйҖҡиҝҮironicеҸҜд»Ҙз»ҷзү©зҗҶжңәпјҢдёҠз”өгҖҒдёӢз”өгҖҒйҮҚеҗҜпјҢиҮӘеҠЁе®үиЈ…ж“ҚдҪңзі»з»ҹпјҢж №жҚ®и®ҫе®ҡзҡ„и§„еҲҷжЁЎжқҝпјҢиҝӣиЎҢиҮӘеҠЁеҢ–й…ҚзҪ®гҖӮironicжҳҜIзүҲејҖе§Ӣиҝӣе…ҘеӯөеҢ–пјҢjзүҲдёҺnovaиҝӣиЎҢйӣҶжҲҗгҖҒKзүҲжӯЈејҸrelease,ironicеҺҹжң¬жҳҜmirantisејҖеҸ‘зҡ„пјҢеҗҺйқўиҙЎзҢ®еҲ°зӨҫеҢәгҖӮйғЁзҪІиҷҡжңәе’ҢйғЁзҪІзү©зҗҶжңәеҜ№еә•еұӮnovaи°ғз”ЁжқҘеҲӣе»әиҷҡжңәпјҢдҪҶеҜ№nova-computeзҡ„еә•еұӮзҡ„hypevisorжҳҜдёҚдёҖж ·зҡ„и°ғз”Ёиҷҡжңәеә•еұӮи°ғз”Ёзҡ„й©ұеҠЁеҸҜиғҪжҳҜlibvirtdгҖҒvmwareapi.VMwareVCDriverзӯүпјҢиҖҢзү©зҗҶжңәеә•еұӮй©ұеҠЁжҳҜironicгҖӮ

з”ЁйҖ”

зӣ®еүҚжңүдәӣжңҚеҠЎеңЁиҷҡжңәдёҠиҝҗиЎҢиҫҫдёҚеҲ°зҗҶжғізҡ„жҖ§иғҪпјҢд»…з®ЎopenstackжӢҘжңүtroveгҖҒsaharaгҖҒmagnumзӯүз»„д»¶з”ЁдәҺж”ҜжҢҒж•°жҚ®еә“е№іеҸ°гҖҒеӨ§ж•°жҚ®е№іеҸ°гҖҒе®№еҷЁе№іеҸ°гҖҒдҪҶжҜ•з«ҹжҳҜеңЁиҷҡжӢҹжңәйҮҢйқўйғЁзҪІжңҚеҠЎпјҢжҖ§иғҪиҝңиҝңдёҚеӨҹгҖӮдҪҶзӣҙжҺҘйғЁзҪІеңЁзү©зҗҶжңәдёҠеҸҲдёҚеҘҪз®ЎзҗҶпјҢжӯӨж—¶ironicеә”иҝҗиҖҢз”ҹдәҶгҖӮironicеҸҜд»Ҙи§ЈеҶізү©зҗҶжңәзҡ„ж·»еҠ гҖҒеҲ йҷӨгҖҒз”өжәҗз®ЎзҗҶгҖҒе®үиЈ…йғЁзҪІгҖҒжңӘжқҘеҸҜиғҪиғҪж”ҜжҢҒиҮӘеҠЁйғЁзҪІж•°жҚ®еә“гҖҒеӨ§ж•°жҚ®е№іеҸ°гҖҒе®№еҷЁе№іеҸ°гҖӮ

ironicз»„д»¶

ironic-apiпјҡиҙҹиҙЈжҺҘж”¶иҜ·жұӮпјҢе№¶дё”е°ҶиҜ·жұӮдј йҖ’з»ҷironic-conductor
ironic-conductorпјҡironicдёӯе”ҜдёҖдёҖдёӘиғҪж №ж•°жҚ®еә“иҝӣиЎҢдәӨдә’зҡ„з»„д»¶пјҢиҙҹиҙЈжҺҘж”¶ironic-apiзҡ„иҜ·жұӮпјҢз„¶еҗҺж №жҚ®иҜ·жұӮжү§иЎҢзӣёеә”зҡ„пјҢеҲӣе»әгҖҒејҖжңәгҖҒе…іжңәгҖҒеҲ йҷӨж“ҚдҪң
ironic-python-agentпјҡйғЁзҪІиЈёжңәж—¶пјҢpxeеҗҜеҠЁиҝӣе…Ҙзҡ„дёҖдёӘbootstrapй•ңеғҸпјҢжӯӨй•ңеғҸжҳҜз”ЁзӨҫеҢәdiskimage-builderе·Ҙе…·еҲ¶еҒҡпјҢй•ңеғҸеҶ…е®үиЈ…дәҶironic-python-agentпјҢиҝӣе…Ҙзі»з»ҹеҗҺдёҺironic-conductorиҝӣиЎҢдәӨдә’пјҢ е°Ҷжң¬ең°зЈҒзӣҳд»Ҙiscsiж–№ејҸжҢӮиҪҪеҲ°жҺ§еҲ¶иҠӮзӮ№пјҢжҺ§еҲ¶иҠӮзӮ№е°Ҷзі»з»ҹddеҲ°иЈёжңәзЈҒзӣҳдёҠ

иЈёжңәйғЁзҪІеҺҹзҗҶ

1пјҢnova bootеҗҜеҠЁдёҖдёӘе®һдҫӢпјҢnova-apiе°ҶиҜ·жұӮйҖҡиҝҮrabbitmqпјҢдј з»ҷnova-conductorпјҢеңЁдј з»ҷnova-schedulerгҖӮ
2пјҢnova-schedulerж №жҚ®дј йҖ’иҝӣжқҘзҡ„flavorе’ҢimageзӯӣйҖүеҮәеҗҲйҖӮзҡ„и®Ўз®—иҠӮзӮ№пјҢз„¶еҗҺе°ҶиҜ·жұӮдј йҖ’еҲ°еҜ№еә”зҡ„и®Ўз®—иҠӮзӮ№гҖӮ
3пјҢnova-computeдёҺironicйҖҡдҝЎпјҢи°ғз”Ёironic-apiгҖӮ
4пјҢironic-apiе°ҶиҜ·жұӮдёӢеҸ‘з»ҷironic-conductorпјҢironic-ductorи°ғз”ЁglanceдёӢиҪҪbootstrapй•ңеғҸгҖӮ
еңЁtftp-sererзӣ®еҪ•з”ҹжҲҗpxelinuxй…ҚзҪ®ж–Үд»¶пјҢе№¶е°Ҷbootstrapй•ңеғҸж”ҫеҲ°tftpзӣ®еҪ•гҖӮ
5пјҢironic-conductorйҖҡиҝҮи°ғз”Ёipmi driverе°Ҷзү©зҗҶжңәејҖжңәпјҢе№¶и®ҫзҪ®д»Ҙpxeзҡ„ж–№ејҸеҗҜеҠЁгҖӮ
6пјҢзү©зҗҶжңҚеҠЎеҷЁеҗҜеҠЁеҗҺпјҢйҖҡиҝҮpxeеҠ иҪҪдәҶbootstrapй•ңеғҸеҗҜеҠЁеҗҺпјҢй•ңеғҸеҶ…ironic-python-agent еҗҜеҠЁпјҢдёҺironic-ductorиҝӣиЎҢдәӨдә’пјҢе°Ҷжң¬ең°зЈҒзӣҳйҖҡиҝҮiscsiж–№ејҸжҢӮиҪҪеҲ°жҺ§еҲ¶иҠӮзӮ№пјҢжҺ§еҲ¶иҠӮзӮ№йҖҡиҝҮddзҡ„ж–№ејҸе°Ҷз”ЁжҲ·йҖүжӢ©зҡ„зі»з»ҹеҶҷе…ҘеҲ°зЈҒзӣҳдёӯгҖӮ
7пјҢеҶҷе…Ҙе®ҢжҜ•еҗҺironicе°Ҷи°ғз”Ёipmi driverеҜ№зү©зҗҶжңҚеҠЎеҷЁиҝӣиЎҢйҮҚеҗҜж“ҚдҪңгҖӮ
8пјҢйҮҚж–°еҗҜеҠЁеҗҺдё»жңәпјҢеҲқж¬Ўиҝӣе…Ҙзі»з»ҹж—¶пјҢиҝҗиЎҢй•ңеғҸеҶ…зҡ„cloud-initиҝӣиЎҢеҲқе§ӢеҢ–пјҢй…ҚзҪ®hostnameе’ҢеҜҶз ҒгҖӮ
иҝҷйҮҢйңҖиҰҒжіЁж„ҸпјҢдҪҝз”Ёзҡ„й•ңеғҸпјҢйңҖиҰҒжҸҗеүҚеңЁ /etc/sysconfig/network-scripts/йҮҢйқўе°ҶзҪ‘еҚЎзҡ„й…ҚзҪ®ж–Үд»¶жҸҗдәӨеҮҶеӨҮеҘҪдёҚз„¶ејҖжңәпјҢзҪ‘еҚЎдјҡupдёҚиө·жқҘгҖӮcentos7йңҖиҰҒдҝ®ж”№grubеҸӮж•°пјҢе°ҶзҪ‘еҚЎеҗҚд»ҘethxжҳҫзӨәпјҢдёҚз„¶д»Ҙи®ҫеӨҮеҗҚжҳҫзӨәпјҢй…ҚзҪ®ж–Үд»¶дёҚеҘҪеҶҷгҖӮ

DockerеҚ•дё»жңәзҪ‘з»ң

2017-11-03 docker docker иҜ„и®ә еӯ—ж•°з»ҹи®Ў: 910(еӯ—) йҳ…иҜ»ж—¶й•ҝ: 4(еҲҶ)

жӯӨж–ҮжЎЈдёәзҝ»йҳ…cloudmanзҡ„жҜҸеӨ©5еҲҶй’ҹзҺ©иҪ¬dockerжҠҖжңҜзҡ„иҜ»д№Ұз¬”и®°гҖӮ

dockerзҪ‘з»ңпјҲеҚ•hostдёҠе®№еҷЁзҪ‘з»ң)
жҳҫзӨәdockerзҪ‘з»ң
docker network list

docker runж—¶йҖҡиҝҮвҖ“networkжҢҮе®ҡзҪ‘з»ң

noneзҪ‘з»ң

noneзҪ‘з»ңе°ұжҳҜд»Җд№ҲйғҪжІЎжңүзҡ„зҪ‘з»ңпјҢжҢӮиҪҪеңЁиҝҷдёӘзҪ‘з»ңзҡ„е®№еҷЁйҷӨдәҶloopbackжІЎжңүе…¶д»–д»»дҪ•зҪ‘еҚЎгҖӮ

дҪҝз”ЁеңәжҷҜпјҡеҜ№дәҺдёҖдәӣе®үе…ЁжҖ§иҰҒжұӮй«ҳдё”дёҚйңҖиҰҒиҒ”зҪ‘зҡ„еә”з”ЁеҸҜд»ҘдҪҝз”ЁгҖӮ

hostзҪ‘з»ң

иҝһжҺҘеҲ°hostзҪ‘з»ңзҡ„е®№еҷЁпјҢе…ұдә«docker е®ҝдё»жңәзҡ„зҪ‘з»ңпјҢе№¶дё”иҝһhostnameд№ҹжҳҜе®ҝдё»жңәзҡ„гҖӮ

дҪҝз”ЁеңәжҷҜпјҡз”ЁhostзҪ‘з»ңжҖ§иғҪжҜ”иҫғй«ҳпјҢе®№еҷЁеҸҜд»ҘзӣҙжҺҘж“ҚдҪңе®ҝдё»жңәзҪ‘з»ңпјҢзјәзӮ№жҳҜзүәзүІдәҶзҒөжҙ»жҖ§пјҢйңҖиҰҒиҖғиҷ‘з«ҜеҸЈеҶІзӘҒй—®йўҳгҖӮ

bridgeзҪ‘з»ң

dockerдҪҝз”ЁжңҖе№ҝжіӣзҡ„ж–№ејҸпјҢdockerе®үиЈ…ж—¶дјҡеҲӣе»әдёҖдёӘdocker0зҡ„linux bridgeпјҢеҰӮжһңдёҚжҢҮе®ҡе®№еҷЁзҪ‘з»ңй»ҳи®ӨйғҪдјҡжҢӮеҲ°docker0дёҠ

еҗҜеҠЁдёӘе®№еҷЁпјҢжҹҘзңӢdocker0
docker run -itd centos_10_23

docker еҶ…зҡ„зҪ‘еҚЎeth0@if28е’Ңlinux bridgeдёҠзҡ„vethfe788c9жҳҜдёҖе Ҷveth-pair

й»ҳи®Өbridgeй…ҚзҪ®зҡ„subnetе°ұжҳҜ172.17.0.3/16зҪ‘е…ідёә172.17.0.1й…ҚеңЁdocker0дёҠпјҢе®№еҷЁеҲӣе»әж—¶дјҡиҮӘеҠЁд»Һ172.17.0.3/16йҮҢйқўжҠҪеҮәдёҖдёӘең°еқҖгҖӮ

user-define(з”ЁжҲ·иҮӘе®ҡд№үзҪ‘з»ң)

user-defineзҪ‘з»ңж”ҜжҢҒпјҡbridgeпјҢoverlayе’ҢmacvlanзҪ‘з»ңй©ұеҠЁпјҢmacvlanе’ҢoverlayеұһдәҺи·Ёе®ҝдё»жңәзҡ„е®№еҷЁзҡ„йҖҡдҝЎпјҢиҝҷйҮҢе…ҲдёҚи®Ёи®әпјҢиҝҷйҮҢи®Ёи®әдҪҝз”Ёbridgeй©ұеҠЁеҲӣе»әuser-defineзҪ‘з»ңгҖӮ

еҲӣе»әдёҖдёӘbridgeдёәdriverзҡ„user-defineзҪ‘з»ң

docker network create вҖ“driver bridge my_net

жҹҘзңӢзҪ‘з»ңзҠ¶жҖҒ

inspectдёҖдёӢпјҢеҸ‘зҺ°ж–°еҲӣе»әзҡ„зҪ‘з»ңй»ҳи®Өsubnetең°еқҖиҮӘеҠЁдјҡд»ҺдёҠдёҖдёӘејҖе§ӢпјҢ

еҲӣе»әеҘҪеҗҺпјҢеӨҡдәҶдёӘзҪ‘жЎҘ

жҢҮе®ҡзҪ‘ж®ө,еҸӘйңҖиҰҒеҠ вҖ“subnet е’ҢвҖ“gatewayеҸӮж•°

е®№еҷЁжҢҮе®ҡipпјҢrunж—¶еҠ дёҠвҖ“ipпјҢеҸӘиғҪжҳҜдҪҝз”Ёзҡ„вҖ“subnetзҡ„зҪ‘з»ңзҡ„е®№еҷЁжүҚиғҪдҪҝз”ЁжҢҮе®ҡipеҠҹиғҪ

е®ҡд№үеҸҰеӨ–дёҖдёӘnetworkзҪ‘з»ң
